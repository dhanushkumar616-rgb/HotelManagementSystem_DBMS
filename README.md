# Hotel Management System

A full-stack Hotel Management System developed using PHP, MySQL, and PDO to streamline hotel operations through automated room booking, guest management, check-in/check-out processing, billing, and payment tracking.

## Features

- Guest Registration and Management
- Room Booking with Real-Time Availability
- Check-In and Check-Out Processing
- Automated Bill Generation
- Payment Recording and Tracking
- Admin Authentication and Dashboard
- Room Status Management
- CRUD Operations for Guests, Rooms, and Bookings
- MySQL Database Integration using PDO

## Technologies Used

- PHP
- MySQL
- PDO (PHP Data Objects)
- HTML5
- CSS3
- XAMPP
- phpMyAdmin

## Project Highlights

- Implements real-world hotel workflow automation.
- Uses PDO prepared statements for secure database operations.
- Maintains room availability through dynamic status updates.
- Supports multi-method payment recording and bill computation.
- Features an administrator dashboard for monitoring hotel activities.

## Database Tables

- Guests
- Rooms
- Bookings
- Payments
- Admins
